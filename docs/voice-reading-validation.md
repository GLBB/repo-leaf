## 已验证能力

- Markdown 朗读提取：Front matter、代码块和 URL 不会进入朗读文本；标题、段落、列表、引用及表格摘要可生成语义段。
- 中文规范化：日期、时间、百分比、金额与常用缩写会自动处理；超长文本按标点分段。
- 系统 TTS：Android 11+ 的 TTS 服务包可见性已声明；真机可发现并调用小米系统中文语音，实际生成非空私有音频。
- 后台播放基础：`Media3 MediaSessionService`、媒体播放前台服务和通知权限均已配置；后台服务在真机成功合成正文缓存。
- 播放状态：文档、段落、段内进度、倍速和音色标识可持久化；正文带稳定段落标识和跟读高亮脚本。
- 无可用音色的降级：TTS 初始化最多等待 8 秒，随后展示“安装离线语音”引导；不会向云端发送文章内容。

## 尚需一次人工设备验收

自动化测试已确认中文合成和后台服务可用。请在手机中打开任意已同步的 Markdown，点击“听全文”，随后：

1. 确认 8 秒内开始出声并显示底部迷你播放器。
2. 锁屏或回到桌面，确认播放持续，并测试播放/暂停、前后段、±5 秒和倍速。
3. 退出后重新打开文章，确认从最近位置恢复。

这三项主要验证厂商系统对通知、锁屏和蓝牙媒体控制的最终呈现，无法由 USB 自动化测试完整替代。
